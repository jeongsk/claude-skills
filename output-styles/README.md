# Output Styles

Claude Code 출력 스타일 모음 플러그인입니다.

## 포함된 스타일

### Korean

모든 응답을 한국어로 제공하는 스타일입니다.

- 설명, 안내, 질문은 한국어로 작성
- 코드, 변수명, 함수명은 영어 유지
- 코드 주석도 한국어로 작성

### Concise

짧고 핵심만 전달하는 간결한 스타일입니다.

- 불필요한 설명 제거
- 인사말, 마무리 문구 생략
- 코드 우선, 설명은 최소화

## 설치

```bash
/plugin install output-styles@jeongsk-claude-skills
```

## 사용법

설치 후 Claude Code 설정에서 원하는 출력 스타일을 선택합니다.
